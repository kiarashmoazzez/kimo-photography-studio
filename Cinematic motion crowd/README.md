# Cinematic Motion Crowd

Reusable AI photo-editing skill for creating a **9:16 black-and-white cinematic wallpaper** from a user portrait.

## Style Summary
A sharp, calm subject stands still inside a fast-moving anonymous crowd. The crowd is heavily motion-blurred while the portrait subject remains recognizable and in focus.

## Files
- `Design.md` — reusable skill / AI instruction file
- `Result 1.jpg` — generated example output

## Best Use
Use this for:
- personal wallpapers
- cinematic profile visuals
- dramatic editorial portraits
- moody black-and-white portrait transformations

## Example Command
```text
Use the attached Design.md file as the full creative direction.
Transform my uploaded portrait into the same cinematic motion-crowd wallpaper style.
Preserve my identity, but recreate the pose, scene, lighting, crowd blur, and black-and-white editorial treatment.
```

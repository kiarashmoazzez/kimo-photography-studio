# Neon Doodle Pop-Art Wallpaper

Reusable AI portrait style for creating a **9:16 vertical neon doodle pop-art wallpaper** from a user portrait.

## Files

- `neon-doodle-pop-art-wallpaper.md` — the full reusable style skill
- `result-image.svg` — visual example / style preview

## How to Use

Upload a portrait and give the AI the Markdown style file. The AI should preserve the person’s identity while recreating the same visual direction: realistic face, relaxed editorial pose, colorful hand-drawn doodle background, neon marker clothing treatment, and phone wallpaper composition.

## Example Command

```text
Use neon-doodle-pop-art-wallpaper.md as the full creative direction. Apply it to my uploaded portrait and create a 9:16 vertical wallpaper in this exact style.
```
